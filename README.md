# Star Player 2

> Protótipo de animações de estrelas/planetas em canvas.

### Estrutura

- `star_player.html` — página principal
- `style.css` — estilos
- `main.js` — inicialização e controle
- `js/scenes/` — cenas (`orbit.js`, `star.js`)

### Como rodar localmente

Use um servidor HTTP local (import de módulos requer servidor):

```bash
python3 -m http.server 8000
```

Abra `http://localhost:8000/star_player.html` no navegador.
